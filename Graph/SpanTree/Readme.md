Difference in Dijkstra and Prims Algorithm

1. Dijkstra's algorithm gives min cost required to visit each vertex of graph. Prim's gives minimum spanning tree which connects all verteces with least possible sum of cost.
2. In Dijkstra's algorithm we use PathSoFar and WeightSoFar. In Prim's we take parentVertext and weight
3. In Dijkstra's algorithm priority queue sort based on WeightSoFar. In Prim's algorithm priority queue sort based on weight of edge
