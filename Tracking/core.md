# Core anchors — DSA hibernation set

A curated ~50-problem list that anchors each NeetCode pattern. These are the problems the weekly queue picker prefers when nothing else is due — the goal is that during a low-activity year you touch every anchor at least once every ~6 months, keeping the pattern warm enough to re-derive everything else on top of it.

_Total: 52 problems across 18 patterns. Generated 2026-07-19 by `curate_core.py`._

## How to edit

- Edit rows in the **Core anchors** table below (add / remove / swap `task` ids).
- Re-run `python Tracking/scripts/curate_core.py` to sync `flags.pinned` in `state.json`.
- Deleting `core.md` and re-running restores the auto-selected default.
